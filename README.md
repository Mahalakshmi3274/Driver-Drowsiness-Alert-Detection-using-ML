# Driver Drowsiness Alert Detection 🚗💤

## 📌 Description
This project detects driver drowsiness using eye (EAR) and mouth (MAR) movements.

## 🛠 Technologies Used
- Python
- OpenCV
- dlib
- Flask

## ⚙️ Features
- Real-time face detection
- Eye closure detection (EAR)
- Yawning detection (MAR)
- Alarm alert system

## ▶️ How to Run
1. Install dependencies
2. Run app.py
3. Open browser
